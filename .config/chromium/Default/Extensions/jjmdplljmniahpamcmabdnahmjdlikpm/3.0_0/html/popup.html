Multi-Forward For Gmail

To use the extension, select the mails to forward and click on the MultiForward icon on the Gmail toolbar (Tutorial).

Please review the extension if you find it useful. For non-Note there is a daily limit of about a 100 mails that you can forward using this extension. This is imposed by Google.
